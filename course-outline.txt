JAVASCRIPT TOPICS FOR REACT JS
 
=================================
1. JavaScript Fundamentals (Must Know)
=================================
 
- Variables (let, const, var)
- Data types (string, number, boolean, null, undefined, object, symbol)
- Operators (arithmetic, comparison, logical)
- Conditional statements (if, switch)
- Loops (for, while, for...of, for...in)
- Functions (declaration, expression)
- Scope (global, function, block)
- Hoisting
- Closures
- this keyword

- Arrow functions (=>)
- Template literals
- Destructuring (array & object)

- sort()

=================================
 
- Object properties & methods
- Object destructuring
- Object.keys()
- Object.values()
- Object.entries()

- Reference vs primitive types
 
 
=================================
5. Asynchronous JavaScript
=================================
 
- Callbacks
- Promises
- async / await
- fetch() API
- Error handling (try...catch)

=================================
6. DOM & Events (Basic Understanding)
=================================
 
- DOM selection
- Event listeners
- Event bubbling & capturing
- Prevent default behavior

- Higher-order functions

8. Modern JavaScript Concepts Used in React

- Closures in hooks
